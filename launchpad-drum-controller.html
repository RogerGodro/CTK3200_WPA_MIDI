Instructions

Connexion: Cliquez sur "Connecter les appareils" pour connecter votre Launchpad MK2 et CTK-3200. L'application peut également fonctionner en mode simulation si les appareils ne sont pas connectés.

Navigation: Utilisez le menu déroulant pour sélectionner un groupe de rythme, ou cliquez directement sur les pads. Basculez entre USER 1 et USER 2 pour accéder aux 32 groupes.

Contrôles Launchpad (si connecté):
• Pads 8×8: Démarrer les rythmes (intro/corps/variation/outro)
• Boutons de droite: 6 premiers = drum sets, 7e = STOP, 8e = PAUSE
• Bouton User 1 (haut): Basculer USER 1 ↔ USER 2
• Bouton User 2 (haut): Tempo +5 BPM
• Bouton Mixer (haut): Tempo -5 BPM
